PORTUGUÊS-BR
-------------------------------------------------------------------------------
Para acrescentar os seus próprios vídeos no tema, os adicione na pasta "videos" no formato .mp4 (até 480p nas Raspberry Pi 3 ou 4).
Vc deverá usar o nome do sistema, todo em minúsculo.
Exemplo:
psx.mp4
nes.mp4

Para acrescentar imagens, as adicione na pasta "images" no formato .jpg usando o mesmo critério de nomes.
Exemplo:
psx.jpg
nes.jpg
-------------------------------------------------------------------------------
ENGLISH
-------------------------------------------------------------------------------
To add your own videos to the theme, add them to the "videos" folder in .mp4 format (up to 480p on Raspberry Pi 3 or 4).
You must use the system name, all lowercase.
Example:
psx.mp4
nes.mp4

To add images, add them to the "images" folder in .jpg format using the same name criteria.
Example:
psx.jpg
nes.jpg
-------------------------------------------------------------------------------
ESPAÑOL
-------------------------------------------------------------------------------
